## Interactive Elements

### Buttons
- **Primary CTA:** Blue background, white text, hover darkens
- **Secondary CTA:** Accent background, white text
- **Outline CTA:** Border only, fills on hover
- **RSVP Button:** Consistent styling with primary CTAs

### Forms
- **Text Input:** Border on focus, ring color matches primary
- **Select Dropdown:** Same styling as text inputs
- **Submit Button:** Full-width, bold text, shadow effect
- **Success Message:** Green background, light text

### Cards
- **Hover Effect:** Shadow increases, subtle lift (transform)
- **Border Radius:** 2xl (rounded corners)
- **Spacing:** Padding 6-8 units, gap between cards

## Animation Effects

### Fade-In (Hero)
- Staggered animation delays (0s, 0.2s, 0.3s, 0.4s, 0.6s)
- Smooth transition over 300ms

### Transitions
- Duration: 300ms standard
- Timing: ease (default)
- Properties: shadow, color, transform, opacity

### Hover States
- Cards: shadow increase
- Links: color change + underline animation
- Buttons: background color + shadow change
